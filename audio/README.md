## Verbe verificate pe PONS (de.pons.com/verbtabellen):
- **sprechen** 🚨 TARE e→i (spricht · sprach · hat gesprochen) — 3 prepoziții: mit (cu cineva) · über+Akk (subiect specific) · von (subiect general)
- **helfen** 🚨 TARE e→i (hilft · half · hat geholfen) — +DAT direct (persoană) + bei + Dat (activitate)
- **gratulieren** REGULAT -ieren (gratulierte · hat gratuliert — fără ge-!) — +DAT persoană + zu + Dat eveniment
- **einladen** 🚨 TARE separabil e→ä (lädt ein · lud ein · hat eingeladen) — +AKK persoană (NU Dat!) + zu + Dat eveniment
